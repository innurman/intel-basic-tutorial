# NumPy
>  [Numpy](http://www.numpy.org/)  : 파이썬이 선형대수학과 같은 계산과학분야에 이용될때 핵심 역할을 하는 라이브러리입니다. Numpy는 고성능의 다차원 배열 객체와 이를 다룰 도구를 제공하므로써 기존보다 훨씬 빠르게 자료를 처리할 수 있게 도와줍니다.
